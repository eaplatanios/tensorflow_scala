  15. def segment[V](xs: Seq[V], n: Seq[Int]): Seq[Seq[V]]

    Segments a sequence according to a provided sequence of segment lengths.

    Segments a sequence according to a provided sequence of segment lengths.

    For example.

    val xs = Seq(3, 5, 2, 77, 12, 45, 78, 21, 89, 1, 0, -1, 123)
val n = Seq(3, 1, 0, 2, 5, 2)
segment(xs, n) = Seq(Seq(3, 5, 2), Seq(77), Seq(), Seq(12, 45), Seq(78, 21, 89, 1, 0), Seq(-1, 123))

    Note that the function returns when either one of xs or n is exhausted. This means that no exception is thrown if the provided segment lengths do not match the original sequence length.

    xs

    Sequence to segment.

    n

    Segment lengths.

    returns

    Sequence containing the segments of xs.
